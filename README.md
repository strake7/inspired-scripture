Source code for [inspiredscripture.com](https://inspiredscripture.com/) — a static site publishing John Edson's Bible study commentaries, built with Next.js and deployed on Netlify.

## Quick Start

```
pnpm install
pnpm start        # dev server at http://localhost:4000
pnpm build        # static export to out/
pnpm test         # Jest test suite
pnpm lint         # ESLint + Prettier
```

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full contributor guide and [ARCHITECTURE.md](ARCHITECTURE.md) for a codebase walkthrough.
